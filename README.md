# MouseTune

扩展鼠标侧键功能

Expand mouse side button functionality

# 使用方法 / Usage Instructions

打开程序，按住侧键的状态下，鼠标滚轮会变成系统音量调节，左右键变成上下曲切换。如果没有进行音量或者曲目切换，点击一下侧键默认为播放/暂停。

When the program is open, holding down the side button will turn the mouse wheel into a system volume control, and the left and right buttons will switch between previous and next tracks. If no volume or track switching is performed, a single click on the side button defaults to play/pause.
